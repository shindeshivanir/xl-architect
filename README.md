# 🟩 XL-Architect
**Master Excel with Precision.** 

[![UI Preview](https://img.shields.io/badge/UI-Modern-success)](#) [![Status](https://img.shields.io/badge/Status-Active_Development-blue)](#)

XL-Architect is an AI-powered web application designed to make Microsoft Excel and spreadsheet management effortless. Whether you are a beginner trying to understand `VLOOKUP` or a data analyst debugging nested `IF` statements, XL-Architect translates plain English into perfectly structured spreadsheet logic.

### ✨ Core Features
*   **🧠 The Architect:** Describe your goal in plain English, and get the exact formula instantly.
*   **🐛 Formula Debugger:** Paste broken or confusing formulas to have the AI deconstruct the complexity and fix syntax errors.
*   **🎨 Logic & Style:** Generate complex dynamic conditioning rules to style your data intelligently.
*   **⌨️ Shortcut Repository:** A searchable vault of precision keyboard commands (toggleable between Mac and Windows).
*   **📚 Visual Templates:** A curated library of pre-engineered data structures and charts.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
